# 삼각형의 내각의 합

## 정의

삼각형의 세 내각의 합은 항상 **180도**입니다.

임의의 삼각형 ABC에서:

$$\angle A + \angle B + \angle C = 180°$$

## 증명

삼각형 ABC의 꼭짓점 A를 지나고 변 BC에 평행한 직선을 그으면:

1. 엇각의 성질에 의해 $\angle B' = \angle B$ (엇각)
2. 엇각의 성질에 의해 $\angle C' = \angle C$ (엇각)
3. 직선 위의 각도이므로 $\angle B' + \angle A + \angle C' = 180°$
4. 따라서 $\angle A + \angle B + \angle C = 180°$

## 외각의 성질

삼각형의 한 외각의 크기는 **이웃하지 않는 두 내각의 합**과 같습니다.

$$\angle ACD = \angle A + \angle B$$

## 활용 예시

- 이등변삼각형에서 두 밑각이 같으므로, 꼭지각을 알면 밑각을 구할 수 있습니다.
- 정삼각형의 각 내각은 $60°$입니다.
- 직각삼각형에서 직각이 아닌 두 각의 합은 $90°$입니다.
