# 직교 개념 축 기반 난이도 프레임워크

## 핵심 원리
문제 난이도 = **축 수** × **연쇄 깊이** × **함정 수**

- 축 수: 동시에 사용하는 독립 개념의 수
- 연쇄 깊이: A결과→B입력 의존 단계 수
- 함정 수: 학생이 빠지기 쉬운 트랩의 수

## 등급 기준

| 등급 | 축 수 | 연쇄 | 함정 | 정답률 |
|------|-------|------|------|--------|
| ★ 기본 | 1개 | 0 | 0 | 80~95% |
| ★★ 응용 | 2개 | 1단계 | 0~1개 | 50~80% |
| ★★★ 심화 | 3~4개 | 2~3단계 | 1~2개 | 15~50% |
| 경시대회 | 4~5개 | 3단계 | 2개+ | 1~15% |

## 조합 유형
- **병렬**: 각 축 독립 계산 후 합침 (비교적 쉬움)
- **연쇄**: A→B→C 순차 의존 (어려움, 오류 증폭)
- **순환**: 조건이 순환하거나 역추론 필요 (초등에서 사용 안 함)

## 함정 유형
1. 등호 경계: 이상/미만 비대칭
2. 동치: 다른 분수가 같은 값으로 약분
3. 경우 누락: 체계적 나열에서 빠뜨림
4. 단위 변환: 시간↔분, cm↔m
5. 개념 전환: 비율→분수→비율 전환
